Support

Some helpful support on using our generated content and web components.

After you have downloaded your web page, you can edit the html file in a text editor and find the text which you want to modify. It is often found inside an element with - class="ref-..."

To combine multiple pages together which you downloaded from Quadflow, simply cut and paste "ref" sections one after the other. "ref" sections are identified by an element with element with - class="ref-...". Note that only web pages from the same category can be combined together. However this can still be done for advanced developers, by adding the script and style tags from the categories you wish to combine.

Download a web page from the page generator app here. Generate the appropriate text content here, then copy the text content into the generated web page using any text editor. Your company/personal info can also be edited and can be found in the element titled "ref-template-data".

Changing the color theme of any web page is very simple, simply navigate to the element - ref-main-theme="enter color here" and enter any color you wish. Most color names will work (e.g. blue, green) and you can also add hex colors. The font can also be easily changed by editing the element ref-main-font="enter font here" and adding any font name inside the double quotes.
